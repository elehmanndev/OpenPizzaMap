#1 in Caiazzo ♥ Saved May 18

Pepe in Grani

●●●●● 4.8 (3,200)
Caiazzo, Italy · Contemporary
Open now

Stamp mechanics: pure CSS — no external assets needed. Each stamp is a double-ring circle (classic passport silhouette), rotated 3–7° in a randomized direction so the page doesn't look machine-printed. Ink color comes from an 8-color palette, picked deterministically per city (hash(citySlug) % 8) — so same city is always the same color, but Naples + Caiazzo + Florence each get their own ink. Two Naples stamps in the demo share the burgundy because that's exactly what would happen in production. City name in heavy display sans (auto-shrinks for names > 8 chars), date in a typewriter serif (Special Elite) for the rubber-stamp feel.

Palette: — pizza red, deep navy, basil green, Spanish ochre, burgundy wine, vintage teal, sepia brown, faded violet. All ink-like, none neon.

If procedural ever feels too thin, the same component swaps cleanly to SVG stamp images — supply 5–10 designs and the CSS class becomes a background-image. Same data shape either way.
